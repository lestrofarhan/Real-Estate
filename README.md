# Real Estate

A modern real estate application built with JavaScript, SCSS, and HTML.

## Overview

This project is a web-based real estate platform designed to showcase properties, manage listings, and provide a seamless user experience for property search and browsing.

## Technology Stack

- **JavaScript** (73.8%) - Core application logic and interactivity
- **SCSS** (25.9%) - Advanced styling and layout
- **HTML** (0.3%) - Markup structure

## Features

- Property listings and search functionality
- Responsive design for desktop and mobile devices
- Modern styling with SCSS
- Interactive user interface with JavaScript

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Node.js and npm (if using a build tool)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/lestrofarhan/Real-Estate.git
```

2. Navigate to the project directory:
```bash
cd Real-Estate
```

3. Open the project in your preferred code editor or run it directly in your browser.

### Usage

Open `index.html` in your web browser to view the application, or set up a local development server:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (http-server)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
